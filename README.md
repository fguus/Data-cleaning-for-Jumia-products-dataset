# Jumia Product Performance Dashboard

An interactive Excel dashboard designed to analyze and visualize product performance on Jumia. The dashboard provides insights into product pricing, discounts, customer reviews, and ratings, helping identify trends, popular products, and areas for improvement.

##  Project Objective

To create an interactive Excel dashboard that enables users to monitor and evaluate the performance of products listed on Jumia through key metrics and visualizations.

##  Key Insights

* Product Pricing Analysis
* Discount Percentage Analysis
* Customer Reviews Analysis
* Product Ratings Analysis
* Category Performance Comparison

##  Dashboard Preview

The dashboard provides a visual overview of product performance, making it easier to identify customer preferences, top-performing products, and market trends.

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting

## How to Use

1. Download the Excel workbook from this repository.
2. Open the workbook in Microsoft Excel.
3. Navigate to the **Dashboard** sheet.
4. Use the slicers and filters to explore the data.
5. Analyze the charts and KPIs to gain insights into product performance.

## Outcome

The dashboard transforms raw product data into actionable insights that support better decision-making and performance analysis.
